**1. DATABASE — Remove 3 tables**

Add 2 new simple tools:
- `write_scratchpad(content)` — overwrites `scratchpad.md` in sandbox
- `read_scratchpad()` — reads `scratchpad.md` from sandbox

**4. AGENT BEHAVIOR — Update system prompt only**

Replace planning instructions with this behavior:

```

1. Write a simple todo list to scratchpad.md
2. Execute todos one by one in a loop

5. Push PR, drop sandbox, clear scratchpad
6. Write a short summary in chat
Do NOT create plan docs. Do NOT break task into phases with separate plan files.
For /docs: use read_file and grep_repo directly on the /docs folder.
```
